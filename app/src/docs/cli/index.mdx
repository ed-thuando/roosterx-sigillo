## Commands

| Command | Description |
|---------|-------------|
| [`login`](./login) | Authenticate to Sigillo with device flow |
| [`logout`](./logout) | Remove saved auth for a scope |
| [`me`](./me) | Show current user info |
| [`setup`](./setup) | Save default project and env for the current directory (stored in ~/.sigillo, not in the repo) |
| [`run`](./run) | Run a command with secrets injected |
| [`secrets`](./secrets) | List secrets for the configured env |
| [`secrets get`](./secrets-get) | Get a secret value |
| [`secrets set`](./secrets-set) | Set a secret value (omit value to read from stdin) |
| [`secrets delete`](./secrets-delete) | Delete a secret |
| [`secrets download`](./secrets-download) | Download all secrets in a chosen format |
| [`orgs`](./orgs) | List organizations |
| [`orgs create`](./orgs-create) | Create an organization |
| [`projects`](./projects) | List projects across organizations |
| [`projects create`](./projects-create) | Create a project |
| [`projects get`](./projects-get) | Get project details |
| [`projects update`](./projects-update) | Update a project |
| [`projects delete`](./projects-delete) | Delete a project |
| [`environments`](./environments) | List envs for the configured project |
| [`environments create`](./environments-create) | Create an env |
| [`environments get`](./environments-get) | Get env details by id or slug |
| [`environments rename`](./environments-rename) | Rename an env by id or slug |
| [`environments delete`](./environments-delete) | Delete an env by id or slug |

## Global Options

| Option | Description |
|--------|-------------|
| `-h, --help` | Display this message |
| `-v, --version` | Display version number |
| `--token [token]` | Auth token override |
| `--api-url [url]` | API URL override (default: https://sigillo.dev) |
