## Commands

| Command | Description |
|---------|-------------|
| [`login`](/cli/login) | Authenticate to Sigillo with device flow |
| [`logout`](/cli/logout) | Remove saved auth for a scope |
| [`me`](/cli/me) | Show current user info |
| [`setup`](/cli/setup) | Save default project and env for the current directory (stored in ~/.sigillo, not in the repo) |
| [`run`](/cli/run) | Run a command with secrets injected |
| [`secrets`](/cli/secrets) | List secrets for the configured env |
| [`secrets get`](/cli/secrets-get) | Get a secret value |
| [`secrets set`](/cli/secrets-set) | Set a secret value (omit value to read from stdin) |
| [`secrets delete`](/cli/secrets-delete) | Delete a secret |
| [`secrets download`](/cli/secrets-download) | Download all secrets in a chosen format |
| [`orgs`](/cli/orgs) | List organizations |
| [`orgs create`](/cli/orgs-create) | Create an organization |
| [`projects`](/cli/projects) | List projects across organizations |
| [`projects create`](/cli/projects-create) | Create a project |
| [`projects get`](/cli/projects-get) | Get project details |
| [`projects update`](/cli/projects-update) | Update a project |
| [`projects delete`](/cli/projects-delete) | Delete a project |
| [`environments`](/cli/environments) | List envs for the configured project |
| [`environments create`](/cli/environments-create) | Create an env |
| [`environments get`](/cli/environments-get) | Get env details by id or slug |
| [`environments rename`](/cli/environments-rename) | Rename an env by id or slug |
| [`environments delete`](/cli/environments-delete) | Delete an env by id or slug |

## Global Options

| Option | Description |
|--------|-------------|
| `-h, --help` | Display this message |
| `-v, --version` | Display version number |
| `--token [token]` | Auth token override |
| `--api-url [url]` | API URL override (default: https://sigillo.dev) |
