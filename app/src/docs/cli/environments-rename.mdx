Rename an env by id or slug

## Usage

```sh
sigillo environments rename <id>
```

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `<id>` | Yes | id |

## Options

| Option | Default | Description |
|--------|---------|-------------|
| `--name [name]` | - | Updated env name |
| `--slug [slug]` | - | Updated env slug |
